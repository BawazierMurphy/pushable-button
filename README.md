## Usage

import 'package:pushable_button/pushable_button.dart';

```dart
PushableButton(
  child: Text(
    'ENROLL NOW',
    style: TextStyle(
      color: Colors.amber,
      fontWeight: FontWeight.bold,
      fontSize: 22,
    ),
  ),
  height: 80,
  elevation: 12,
  color: Colors.white,
  borderRadius: 12,
  onPressed: () {},
);
```
